##### Установка

Для успешной работоспособности, убедитесь, что у вас установлен [![NPM Version][npm-logo]][npm]

```console
npm install
```

##### Запуск

```console
npm run start
```

Для запуска проекта в режиме разработки используйте

```console
npm run dev
```
